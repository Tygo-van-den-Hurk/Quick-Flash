# Security Policy

## Reporting a Vulnerability

If you find a security issue, please sent me an email instead of opening an issue. You can find the email and GPG key [on my GitHub profile](http://redirects.tygo.van.den.hurk.dev/github/personal). Do **not** open a public issue until the vulnerability has been resolved and disclosed.
